Weather App

Created a Weather app using which users can search for temperature of different city.

Motivation

Developed it while completing the course "The Complete 2022 Web Development Bootcamp" on Udemy. Aim was to learn how APIs work and how to connect API using NodeJS.

Tech/Framework used

HTML, CSS, ExpressJS, NodeJS, OpenWeather Map API.

Installation

Install all the dependencies by typing the command "npm install" inside command line.

Tests

Enter the name of the city inside the search box and press go button.
